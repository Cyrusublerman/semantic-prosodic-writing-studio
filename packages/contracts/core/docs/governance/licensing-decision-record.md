# Licensing Decision Record

Status: owner decision required. This record intentionally does not select a licence.

## Scope requiring an explicit decision

- Python source code;
- JSON Schemas;
- fixtures and examples;
- generated TypeScript declarations;
- documentation and normative specification;
- release archives and manifests;
- external contributions.

## Options to evaluate

1. Proprietary internal use only.
2. Source-available with restricted commercial or redistribution rights.
3. Permissive open source for code and data-compatible terms for schemas/fixtures.
4. Copyleft open source.
5. Mixed licensing by artifact class.

## Decision criteria

- intended public distribution and collaboration;
- downstream use by SPWS and other PKL tools;
- ability to redistribute generated artifacts and fixtures;
- third-party dependency compatibility;
- patent and trademark position where relevant;
- contribution ownership and contributor terms;
- whether schemas should be usable independently of implementation code.

## Required decision output

The owner-approved record must state:

- selected licence for each artifact class;
- copyright holder;
- effective date;
- contribution policy;
- compatibility rationale;
- whether earlier prereleases are relicensed;
- files added or changed to implement the decision.

Until this is approved, `A-LIC-001` remains blocking and no external or stable distribution claim may imply permission not actually granted.
